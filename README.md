# ForbidAd4TieBa

一个轻量级的开源 Xposed 模块，用于去除百度贴吧的各种多余广告并净化页面体验   
**我的-设置图标（长按）进入模块设置**

## 功能特性
- 支持反混淆
- 屏蔽开屏广告
- 屏蔽信息流广告：屏蔽首页推荐列表的广告卡片及App推广
- 屏蔽评论区广告：屏蔽评论区的广告卡片及App推广
- 屏蔽直播内容：屏蔽首页直播卡片
- 屏蔽推荐进吧卡片：屏蔽首页推荐进吧卡片
- 首页Tab精简：仅保留推荐Tab
- 底部Tab精简：去除小卖部Tab
- 进吧页面精简：去除推荐列表

## 兼容性
- 贴吧客户端版本兼容测试：`22.x.x.x正式版` 
- 其它版本自行测试
