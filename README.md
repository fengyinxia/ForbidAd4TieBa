# ForbidAd4TieBa

一个轻量级的开源 Xposed / LSPosed 模块，用于去除百度贴吧的多余广告并净化页面体验。  
**我的 → 设置图标（长按）进入模块设置**

## 功能特性
- 支持反混淆与符号扫描缓存
- 首次进入宿主且符号未解析时，会自动弹出扫描窗口
- 支持手动反混淆扫描、扫描日志查看、复制日志、扫描后重启宿主
- 屏蔽开屏广告
- 屏蔽信息流广告：屏蔽首页推荐列表的广告卡片及 App 推广
- 屏蔽评论区广告：屏蔽评论区的广告卡片及 App 推广
- 屏蔽直播内容：屏蔽首页直播卡片
- 首页顶栏净化：隐藏首页顶栏按钮、Tab 和搜索栏，并禁止左右滑动切换首页 Tab
- 底部 Tab 精简：去除小卖部 Tab
- 个人页面净化：隐藏我的页面中的会员主卡、游戏专区、横幅 Banner、钱包/度小满卡片
- 进吧页面净化：合并推荐进吧卡片过滤与进吧 WebView 页面精简
- 默认启动进入进吧页

## 使用说明
- 安装后在 LSPosed / Xposed 中启用模块
- 作用域为：`com.baidu.tieba`
- 进入贴吧宿主后：
  - 如果本地已有可用符号缓存，会直接按缓存工作
  - 如果当前版本未命中可用缓存，首次进入宿主会自动弹出扫描窗口
- 需要重新扫描时，可在贴吧内：
  - 进入 **我的** 页面
  - **长按设置图标** 打开模块设置
  - 点击 **手动反混淆**

## 当前可配置项
- 屏蔽广告
- 屏蔽直播内容
- 屏蔽首页 Tab（仅保留推荐）
- 屏蔽小卖部 Tab
- 个人页面净化
- 进吧页面净化

## 兼容性
- 当前版本号：`26031905 (versionCode 8)`
- 贴吧客户端版本兼容测试：`22.x.x.x 正式版`
- 其它版本自行测试
- 若当前版本混淆变化较大，可先使用自动扫描或手动反混淆功能

## 说明
- 本模块使用宿主侧 SharedPreferences 缓存符号结果
- 当贴吧版本更新后，旧缓存可能失效，此时会重新进入扫描流程
- 若扫描结果为 `partial` 或 `unsupported`，设置页会提示部分功能不可用
